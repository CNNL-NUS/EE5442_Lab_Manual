Installation on Windows

To install mumax3 on your on standalone machine in Windows you can follow the below mentioned steps. Primarily you will need an NVIDIA GPU with compute capability 5.0 or higher, as listed here. Additionally you will need to use NVIDIA's proprietary graphics driver, which may already be installed on your system.

Check the Nvidia Driver version

  1. Go to run menu on your computer from the search bar, and run cmd.
  1. In the command line, key in “nvidia-smi” and hit enter, to gather information on the driver version. This would be required to choose the MuMax3 package version to be downloaded for installation.

Downloading mumax3

  1. You can download mumax3 on their official webpage https://mumax.github.io/download.html.

    Alternatively you can visit their GitHub page to download or edit the source for any updates.

  1. Select your OS platform Windows or Linux.

  1. Select NVIDIA driver based on your device and click Download.

Install mumax3

  1. Unzip the downloaded mumax3.10_windows_cudaXX.X.zip file.

  1. Right click This PC and click Properties.

  1. Click Advanced system settings.

  1. Select Environment Variables and click edit

  1. Click New (1) and Browse (2).

    Select the folder where mumax3 is installed and click OK.

Verify Installation

  1. Type in “mumax3” in Command Prompt, and it will show this if you successfully install mumax3.

  1. Double click the mumax3.exe, and a window and a web GUI will pop up.

Executing mumax scripts

  1. Method 1: Type in “mumax3 myfile.mx3” in Command Prompt.

  1. Method 2: Right click the XX.mx3 script file → Open with and select mumax3.exe open and execute it.
